# BiScatter

Demonstation code for BiScatter downlink communication

## File Structure
Root Folder\
|-BiScatter_analysis.ipynb: main analysis file\
|-collect_data.py: file used to collect data\
|-utils.py: contains utility functions\
|-requirement.txt\
|-Data: folder contains example dataset\
|-Hardware: folder contains hardware setup

## Running the code
Run "BiScatter_analysis.ipynb" to start with.\
Example dataset uploaded under "./Data/"\
Run "collect_data.py" to collect data
